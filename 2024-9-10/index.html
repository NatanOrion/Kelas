Belajar Html dan css

Html digunakan untuk membuat tampilan website. Css digunaka untuk mengatur tampilan

Materi Html

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • Table untuk tabel
  • input untuk form
  • li untuk list atau data

Materi css

  1. Margin, padding, position untuk mengatur jarak anatara elemen html
  2. display untuk mengatur elemen html agar tampil atau tidak tampil, mendatar(kolom), atau menurun (baris)
  3. style untuk mengantur font
